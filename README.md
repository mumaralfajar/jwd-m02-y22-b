## Requirements

* XAMPP/Laragon
* PHP >= 7.0.0
* Browser : Google Chrome/Mozilla Firefox/Opera
* Bootstrap >= 5.2.0

## Instalation

1. Letakan folder di dalam
   > *...\htdocs*  (XAMPP)
   > 
   > atau
   > 
   > *...\www*  (Laragon)
2. Jalankan *XAMPP -> Apache* atau *Laragon -> Start All*.
3. Buka Browser ketikan
   > *localhost/[nama-folder]* (XAMPP)
   > 
   > atau
   > 
   > *[nama-folder].test* (Laragon)
